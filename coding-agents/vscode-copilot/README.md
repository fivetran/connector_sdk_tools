# VS Code + GitHub Copilot Setup

### Automatic (coming soon)

```bash
fivetran ai setup --agent vscode-copilot
```

### Install script

```bash
bash scripts/install-vscode-copilot.sh /path/to/my-connector
```

Copies `coding-agents/AGENTS.md` into your connector project root. Copilot will pick it up automatically.

### Manual

```bash
cp coding-agents/AGENTS.md /path/to/my-connector/AGENTS.md
```
